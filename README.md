# mirrorfree-filer

A Python package scaffold for filing dormant LLP accounts for Mirror Free LLP (Companies House number: OC444103).

## Structure

- `src/mirrorfree_filer/` contains the installable package modules.
- `tests/` contains basic regression tests.

## Development

Install in editable mode:

```bash
python3 -m pip install -e .[dev]
```
